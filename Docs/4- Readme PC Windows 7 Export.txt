Exporting your game for PC is VERY EASY, just follow the steps below.

##################################################################################

1) Start "UnrealFrontendPackagerPC" shortcut located at the folder "C:\UDKUltimate\UDKUltimate Desktop Shortcuts". Once opened UDKFrontend.exe, you click on the tab Cooking, then type the complete name of the map (UDK-UDKUltimate.udk), set the option Full Recook, then you click on the button Make (click the little black arrow and select full recompile). All the other options are already setup.

##################################################################################

2) After the compiling scripts process finishes, you click on Cook, then after the cooking finishes, you click on Package Game. After this a window will popup asking to you type a name for your Game. So after this, you will see on the root folder an executable file (UDKInstall-MyGameUDK). You install this executable to any place of your computer (i.e C:\MyGameUDK). Then you copy the folder C:\UDKUltimate\Binaries\Win32 or/and C:\UDKUltimate\Binaries\Win64 to C:\MyGameUDK\Binaries, and overwrite the files there. Now you can share with others your game which is this folder C:\MyGameUDK\. You can simply compress and share the folder, because there will be no need for installation of the game, it`s portable, everything it needs to run is already inside the game folder "C:\MyGameUDK\".

##################################################################################

3) Have fun playing your new PC game :)
